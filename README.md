## 🌟 Featured Projects

- **Distributed AI Invoice Auditor**
  A distributed multi-agent AI invoice auditor (Google ADK + LangGraph) where 3 agents and 9 tools coordinate over MCP and a custom Agent-to-Agent (A2A) protocol to execute a 7-step pipeline with automatic fallback.
  - *Stack:* Google ADK, LangGraph, Azure OpenAI (GPT-4o), LiteLLM, Langfuse, Presidio PII

- **Hybrid-Retrieval RAG Chatbot**
  A 4-node LangGraph-based chatbot implementing hybrid-retrieval (FAISS + BM25, 65/35 weighting) with retrieve, re-rank, synthesize, and self-evaluate steps to answer queries over an ERP database.
  - *Stack:* LangGraph, FAISS, BM25, SQLite

- **Polling Service (Microservices)**
  A Django REST polling microservice with a Kafka-driven, event-based voting pipeline that records votes through dedicated consumers, shielding the request path from viral write spikes.
  - *Stack:* Django REST Framework, FastAPI, PostgreSQL, Kafka, JWT, RBAC, API Gateway, Docker Compose, pytest

- **Users-Module**
  User authentication & authorization microservice — plug-and-play for microservice stacks.
  - *Stack:* FastAPI, PostgreSQL, Docker, JWT, OAuth2, RBAC

- **Stars (Kharchakhata)**
  A web app for visualizing daily expenses — track, organize, and gain insight into your spending.
  - *Stack:* Flask, SQLAlchemy, SMTP mail, Dash visualizations

- **URL_Shortner**
  Simple, fast web service for secure, customizable URL shortening services.
  - *Stack:* FastAPI, Python, authentication, CRUD

- **Blogit**
  Django-based blog platform — full feature set from authentication to bookmarks, likes, subscriptions, and CRUD.

- **My-Portfolio**
  Portfolio website — showcase of work, experience, and recommendations.
  - *Stack:* Django, Python, HTML
